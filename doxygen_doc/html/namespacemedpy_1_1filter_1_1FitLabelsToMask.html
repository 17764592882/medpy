def medpy.filter.FitLabelsToMask.fit_labels_to_mask (   image_labels,
  image_mask 
)

Reduces a label images by overlaying it with a binary mask and assign the labels either to the mask or to the background.

The resulting binary mask is the nearest expression the label image can form of the supplied binary mask.

Parameters:
image_labels,:A labeled image, i.e., numpy array with labeled regions.
image_mask,:A mask image, i.e., a binary numpy array with False for background and True for foreground.
Returns:
: A mask image, i.e. a binary numpy array with False for background and True for foreground.

Note that the input images must be of the same shape, offset and physical spacing.
